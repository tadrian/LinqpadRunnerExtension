# LinqPad Runner for VS Code

Execute `.linq` files directly in VS Code using LinqPad's LPRun tool.

## Features

- Real-Time Streaming Results Viewer: JSON trees, sortable tables
- Live output updates with stop/play controls
- C# Syntax Highlighting
- F12 Go to Definition
- One-Click Execution
- Smart Editor Group Management
- Output Destination Control
